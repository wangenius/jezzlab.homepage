# 设定

设定是介子中核心概念之一，设定是世界观的基础。设定是随时间有不变的属性、概念等。你不仅可以用设定给这个世界增添规则，同时也可以利用设定给一些角色、物品等对象增添不随时间变化的固有属性。

与情节、角色不同的是，设定是静态的，不会随着情节（时间）的发展而变化。

例如：

1. 武侠、玄幻世界中的等级。
2. 古代世界中的官职等。
3. MBTI 等性格测试。
4. 科幻小说中，物理学中的相关设定。
5. 等等。

## 用途

1. 用于角色状态属性中的选择。
2. 在生成情节、角色、以及小说写作过程中的上下文参考。
